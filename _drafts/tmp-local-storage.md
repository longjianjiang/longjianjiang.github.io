# 持久化需求

## KV存储

KV存储一般使用UserDefaults，常见场景是一些标记位。

## KeySet

这种使用KV的话就是一个map的结构, {string: <set>}。

这种一般和业务有关系，所以最好还是存库比较好一些，可以建一张表，有两个字段:

 +-----------+-----------+
 | key       | set       |            
 +-----------+-----------+
 | key1      | "1,2,3"   |
 +-----------+-----------+

set 是用set转成字符串进行存。
