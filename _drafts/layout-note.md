# AutoLayout

## Masonry

Masonry 封装了`NSLayoutConstraint`的接口，提供了链式调用的语法。

## NSLayoutAnchor

iOS9 Apple提供了一种快捷的方式进行创建约束，如下所示:

{% highlight objc %}
NSLayoutConstraint *constraint = [myView.topAnchor constraintEqualToAnchor:otherView.topAnchor constant:10];
{% endhighlight %}
